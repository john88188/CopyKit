# CopyKit | 摘抄猫 v0.3.0 (English)

## Community Build Overview

This directory is for community distribution and contains extension artifacts and public docs.

## Download & Verify

- Build package: `build/copykit-v0.3.0.zip`
- Checksum: `build/copykit-v0.3.0.zip.sha256`

## Install (Developer Mode)

1. Unzip `build/copykit-v0.3.0.zip`
2. Open `chrome://extensions/` or `edge://extensions/`
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select the extracted folder (must contain `manifest.json` at root)
