# CopyKit | 摘抄猫 v0.3.1（中文）

## 社区版说明

本目录用于社区分发，包含插件构建物与公开文档。

## 下载与校验

- 构建包：`build/copykit-v0.3.1.zip`
- 校验值：`build/copykit-v0.3.1.zip.sha256`

## 安装（开发者模式）

1. 解压 `build/copykit-v0.3.1.zip`
2. 打开 `chrome://extensions/` 或 `edge://extensions/`
3. 开启“开发者模式”
4. 点击“加载已解压的扩展程序”
5. 选择解压目录（目录内需直接看到 `manifest.json`）
